## 簡介
送養者可以透過此平台將寵物資訊發布於領養頁面；領養者可以於領養頁面找尋合適的寵物並獲得送養者聯絡方式。

## 網頁架構
### 前端
- 形象首頁
- 登入頁面
- 領養頁面
- 送養頁面
- 寵物頁面
### 後端
- 登入系統
  - 客戶端認證
  - 客戶資料新增
- 寵物資料新增
## 使用技術
- React
- JavaScript
- Node.js
- API
- Express
- MongoDB
